# **App Name**: BudgetView

## Core Features:

- User Authentication: User registration and login using Firebase Authentication with email and password.
- Dashboard Summary: Dashboard displaying current month's total income, expenses, and remaining balance.
- Transaction Input: Form for adding new transactions with fields for description, amount, date, type, and category.
- Transaction List: List view of all transactions for the month, with options to edit or delete transactions.
- Custom Categories: Allow user to create edit, and delete categories. 
- Report Date Filter: Display filters to let user specify which date ranges to display.
- Transaction categorization suggestions: LLM tool will offer to suggest an appropriate transaction category for each expense, based on a description. 

## Style Guidelines:

- Primary color: Muted green (#90EE90) to evoke financial health.
- Background color: Light gray (#F0F0F0) for a clean and modern feel.
- Accent color: Soft blue (#ADD8E6) to highlight important information and actions.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and readable interface.
- Use simple, clear icons for categories and actions.
- Prioritize a clean and intuitive layout with a clear hierarchy.
- Use subtle animations for transitions and feedback.
